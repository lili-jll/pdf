Dealer/Customer direct ship process.

This document provides a flow of the direct ship process of supplying parts to a customer/dealer direct from the supplier (as opposed to shipping from the SAIC warehouse).

Step 1.

Define the item number as direct ship in menu option IC403.

This only needs to be done once.

Enter a reorder code of E to define the part as emergency direct supply.

 

Step 2.

Enter the order in OE202 (or dealer connect DC202) as normal. Since there is no stock for the item it will go on backorder and be tagged as direct supply.

The order will appear in OE209.

 

Step 3.

Run menu option OEA06 periodically to pick all the outstanding direct ship orders.

This menu option will create the supplier PO.  It will also show as ¡°Stock On Order¡± IC602.

-          Enter the PO type as D,

-          Enter the supplier code,

The PO¡¯s will have a PO number of e.g. EM00000001

 

Step 3a.

Print the PO in PU202 using the print option.

This will flag the PO as printed.

 

Step 4.

When the supplier invoice is received, receipt the supplier invoice in SH204 similar to any other supplier invoice.

The items can be loaded using option 8 ¨C Retrieve P/O.  Using option 8 will load all the details from the direct ship PO correctly.

 

 

Step 5.

Receipt the shipment in SH204 using option S ¨C Simple Receipt.

This step will close off the original backorder, receipt the parts into stock and then invoice the customer/dealer.
